Welcome to Ngāi Tahu Data

Kia ora! Welcome to the Ngāi Tahu Data Tool, a detailed resource for data related to Ngāi Tahu. This tool provides valuable insights into current state of the Ngāi Tahu nation data.

Getting Started

Select a Topic

Browse Topics – Start by exploring a range of topics from the main menu. Each topic is equipped with detailed charts and datasets tailored to provide you with thorough insights.

View Charts – As you select a topic, you will be navigated to visual representations of your data. These charts are interactive, you can also download the raw data or image.

Chat with Ngai Tahu Data Assistant:

Start a conversation – Interact with the Data Assistant by asking a question about any topic displayed in the left navigation panel. The Data Assistant will provide insights and analysis on Ngāi Tahu data
